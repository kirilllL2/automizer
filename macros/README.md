# Макросы в Automizer

Макросы в Automizer определяются как Python-файлы в папке `macros/`. Каждый макрос - это отдельный `.py` файл с функцией `run()`.

## Структура макроса

```python
# Метаданные (отображаются в UI)
NAME = "Название макроса"
DESCRIPTION = "Описание того, что делает макрос"

# Импортируем вспомогательные функции (только для Windows)
from src.macro import click, delay, wait_for


def run():
    """
    Основная функция макроса.
    Вызывается при запуске из UI.
    """
    # Ваш код здесь
    click(100, 100)
    delay(1.0)
```

## Доступные функции

### `click(x: int, y: int)`
Выполняет клик в указанную точку экрана.

**Параметры:**
- `x` - координата X
- `y` - координата Y

**Пример:**
```python
click(500, 300)  # Клик в точку (500, 300)
```

### `delay(seconds: float)`
Выполняет задержку на указанное количество секунд.

**Параметры:**
- `seconds` - длительность задержки в секундах

**Пример:**
```python
delay(2.5)  # Задержка на 2.5 секунды
```

### `wait_for(condition: Callable[[], bool], timeout: float = 10.0, interval: float = 0.1) -> bool`
Ждет выполнения условия до истечения таймаута.

**Параметры:**
- `condition` - функция без аргументов, возвращающая True/False
- `timeout` - максимальное время ожидания в секундах (по умолчанию 10.0)
- `interval` - интервал проверки условия в секундах (по умолчанию 0.1)

**Возвращает:**
- `True` если условие выполнилось до истечения таймаута
- `False` если таймаут истек

**Пример:**
```python
# Ждем появления окна в течение 30 секунд
def window_exists():
    # Проверка существования окна
    return True  # или False

if wait_for(window_exists, timeout=30.0):
    print("Окно найдено!")
else:
    print("Окно не найдено")
```

## Создание нового макроса

1. Создайте новый файл в папке `macros/` с именем `my_macro.py`
2. Добавьте метаданные и функцию `run()`:

```python
NAME = "Мой макрос"
DESCRIPTION = "Описание моего макроса"

from src.macro import click, delay


def run():
    # Ваша логика здесь
    click(100, 100)
    delay(0.5)
    click(200, 200)
```

3. Обновите список макросов в UI (кнопка "🔄 Обновить")
4. Выберите макрос и нажмите "▶️ Запустить"

## Примеры использования

### Простая последовательность кликов
```python
NAME = "Двойной клик"
DESCRIPTION = "Клик в две точки"

from src.macro import click, delay


def run():
    click(100, 100)
    delay(0.1)
    click(100, 100)
```

### Цикл с задержкой
```python
NAME = "Многократный клик"
DESCRIPTION = "Кликает 5 раз в одну точку"

from src.macro import click, delay


def run():
    for i in range(5):
        click(500, 300)
        delay(0.5)
        print(f"Клик {i+1} из 5")
```

### Ожидание условия
```python
NAME = "Ожидание и клик"
DESCRIPTION = "Ждет появления элемента и кликает"

from src.macro import click, delay, wait_for


def run():
    # Ждем 10 секунд
    if wait_for(lambda: True, timeout=10.0):
        click(400, 400)
        print("Успешно!")
    else:
        print("Таймаут")
```

## Примечания

- Макросы выполняются в отдельном потоке, поэтому UI остается отзывчивым
- Для остановки выполняющегося макроса используйте кнопку "⏹️ Остановить"
- Файлы макросов, начинающиеся с `_`, игнорируются (например, `_utils.py`)
- На Windows для работы функций `click()` требуется установленный `pywin32`
