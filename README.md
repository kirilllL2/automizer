# Automizer

Современное десктопное Python приложение для автоматизации задач с красивым PyQt5 интерфейсом.

## Особенности дизайна

- 🎨 **Градиентные фоны и элементы** - современные цветовые переходы
- ✨ **Тени и эффекты свечения** - глубина и объем интерфейса
- 🔘 **Овальные кнопки** с анимацией при наведении
- 📱 **Адаптивная резиновая верстка** - подстраивается под любой размер окна
- 🌊 **Плавные переходы** между состояниями элементов

## Структура проекта

```
automizer/
├── src/              # Исходный код приложения
│   ├── config.py     # Модуль конфигурации
│   ├── ui/           # UI модуль (PyQt5 интерфейс)
│   │   └── app.py    # Главное приложение с современным дизайном
│   ├── normalizer/   # Модуль нормализации окон
│   ├── screenshot/   # Модуль работы со скриншотами
│   ├── storage/      # Хранилище пресетов и данных
│   └── window_manager.py  # Управление окнами
├── config/           # Файлы конфигурации
│   └── default.yaml  # Настройки по умолчанию
├── tests/            # Тесты
├── pyproject.toml    # Зависимости и настройки проекта
└── README.md         # Этот файл
```

## Возможности

### Нормализация окон
- Выбор процесса из списка с поиском
- Настройка позиции и размера окна
- Предустановленные размеры (половина экрана, четверти и т.д.)
- Сохранение настроек как пресеты

### Управление пресетами
- Создание, редактирование и удаление пресетов
- Поиск пресетов по названию
- Применение пресетов к найденным окнам

### Работа со скриншотами
- Выделение области экрана с предпросмотром
- Предустановленные размеры (половина экрана, четверти, полный экран)
- Привязка скриншотов к пресетам процессов
- Название и описание для каждого скриншота
- Поиск и управление скриншотами

## Установка зависимостей

Проект использует `uv` для управления зависимостями:

```bash
# Установить uv (если не установлен)
pip install uv

# Установить все зависимости
uv sync

# Установить зависимости для разработки
uv sync --dev
```

### Основные зависимости

- **PyQt5** - современный GUI фреймворк
- **PyYAML** - работа с конфигурацией
- **Pillow** - обработка изображений
- **pywin32** - управление окнами (только Windows)

## Конфигурация

Приложение поддерживает многоуровневую конфигурацию:

1. **Значения по умолчанию** - `config/default.yaml`
2. **Пользовательские настройки** - `~/.automizer/config/config.yaml`
3. **Переменные окружения** - префикс `AUTOMIZER_`

### Пример пользовательской конфигурации

Создайте файл `~/.automizer/config/config.yaml`:

```yaml
app:
  debug: true
  log_level: "DEBUG"

ui:
  theme: "dark"
  window:
    width: 1400
    height: 900
```

### Переменные окружения

```bash
export AUTOMIZER_APP_DEBUG=true
export AUTOMIZER_UI_THEME=dark
```

## Запуск

```bash
# В режиме разработки
uv run python -m src.ui.app

# После установки
uv pip install -e .
automizer
```

## Разработка

```bash
# Запустить тесты
uv run pytest

# Проверка кода
uv run ruff check src/
uv run black --check src/
uv run mypy src/

# Форматирование кода
uv run black src/
uv run ruff check --fix src/
```

## Требования

- Python >= 3.10
- uv (менеджер пакетов)
- PyQt5 >= 5.15.0
- Для Windows: pywin32

## Скриншоты интерфейса

Приложение использует современный темный дизайн с:
- Градиентными кнопками (фиолетовый, розовый, голубой)
- Карточками с тенями и закругленными углами
- Плавными hover-эффектами
- Эргономичным расположением элементов
