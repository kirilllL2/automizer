# Automizer

Современное десктопное приложение на PyQt5 для автоматизации управления окнами и создания скриншотов.

## ✨ Особенности

- **Современный UI/UX дизайн** с градиентами, тенями и овальными кнопками
- **Темная тема** в стиле Tokyo Night
- **Интуитивный интерфейс** с вкладками для различных функций
- **Быстрый захват** скриншотов по пресетам

## Структура проекта

```
automizer/
├── src/              # Исходный код приложения
│   ├── config.py     # Модуль конфигурации
│   ├── ui/           # UI модуль (PyQt5 интерфейс)
│   │   └── app.py    # Главное приложение
│   ├── normalizer/   # Модуль нормализации окон
│   ├── screenshot/   # Модуль работы со скриншотами
│   ├── storage/      # Хранилище пресетов и данных
│   └── window_manager.py  # Управление окнами
├── config/           # Файлы конфигурации
│   └── default.yaml  # Настройки по умолчанию
├── tests/            # Тесты
├── pyproject.toml    # Зависимости и настройки проекта
└── README.md         # Этот файл
```

## Возможности

### 🪟 Нормализация окон
- Выбор процесса из списка с поиском
- Настройка позиции и размера окна
- Предустановленные размеры (половина экрана, четверти и т.д.)
- Сохранение настроек как пресеты
- Применение пресетов к найденным окнам

### 💾 Управление пресетами процессов
- Создание, редактирование и удаление пресетов
- Поиск пресетов по названию
- Применение пресетов к найденным окнам
- Двойной клик для быстрого применения

### 📸 Работа со скриншотами
- Выделение области экрана с настройкой координат и размеров
- Предустановленные размеры (половина экрана, четверти, полный экран)
- Привязка скриншотов к пресетам процессов
- Название и описание для каждого скриншота
- Поиск и управление скриншотами
- Открытие файла скриншота или папки со скриншотами

### 🎯 Пресеты скриншотов
- Быстрый захват: нормализация + скриншот по пресету
- Создание/редактирование/удаление пресетов
- Привязка к пресету процесса
- Настройка пути сохранения
- Настройка области захвата

## Установка зависимостей

Проект использует `uv` для управления зависимостями:

```bash
# Установить uv (если не установлен)
pip install uv

# Установить все зависимости
uv sync

# Установить зависимости для разработки
uv sync --dev
```

### Для Windows

Приложение использует PyQt5 и pywin32 для работы с окнами:

```bash
# PyQt5 будет установлен автоматически
# pywin32 требуется только для Windows
```

## Конфигурация

Приложение поддерживает многоуровневую конфигурацию:

1. **Значения по умолчанию** - `config/default.yaml`
2. **Пользовательские настройки** - `~/.automizer/config/config.yaml`
3. **Переменные окружения** - префикс `AUTOMIZER_`

### Пример пользовательской конфигурации

Создайте файл `~/.automizer/config/config.yaml`:

```yaml
app:
  debug: true
  log_level: "DEBUG"

ui:
  theme: "dark"
  window:
    width: 1400
    height: 900
```

### Переменные окружения

```bash
export AUTOMIZER_APP_DEBUG=true
export AUTOMIZER_UI_THEME=dark
```

## Запуск

```bash
# В режиме разработки
uv run python -m src.ui.app

# После установки
uv pip install -e .
python -m src.ui.app
```

## Дизайн и UX

Приложение использует современный дизайн с:

- **Градиентные кнопки** с эффектом при наведении
- **Тени** для создания глубины интерфейса
- **Овальные кнопки** для приятного визуального восприятия
- **Темная тема** Tokyo Night для снижения нагрузки на глаза
- **Адаптивная верстка** с изменяемым размером окна
- **Всплывающие уведомления** о результатах операций

### Цветовая палитра

- Основной фон: `#1a1b26`
- Вторичный фон: `#24283b`
- Акцентный цвет: `#7aa2f7` (синий) → `#bb9af7` (фиолетовый) градиент
- Текст: `#c0caf5`

## Требования

- Python >= 3.10
- PyQt5 >= 5.15.0
- Pillow >= 10.0.0
- pywin32 >= 306 (только для Windows)
- uv (менеджер пакетов)

## Платформы

- **Windows**: Полная поддержка (через pywin32)
- **Linux**: Базовая поддержка (требуется адаптация window_manager)
- **macOS**: Не поддерживается (требуется адаптация)
